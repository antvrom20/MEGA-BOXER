\\WELCOME TO MEGA BOXER V1.0//



Welcome to Mega Boxer!
The goal of the game is to push and hit the other player out of the arena! The first to do so wins!
Here's Some controls to get you started:
WASD: Move
Mouse: Look
K: Punch
Space: Secret...

Originally, the was intended to have a health based system, but we simply couldn't add this feature as well as a planned mixed reality mode
to the game and full online support, due to a misunderstanding of the complexity of these features. Hopefully, in a later update and with more understanding,
we can add these features back in!

Feel free to make sugesstions and report bugs to us!
We really hope you do enjoy our game!
